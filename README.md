### 🧠 About Me

I'm a Computer Science undergrad at **FAST NUCES Islamabad**, specializing in AI engineering, currently interning as an **AI Agentic Intern at the NCRA-UAV Dependability Lab**, where I work on agentic systems and LangGraph based automation for a cost optimization initiative.

Outside of coursework and the internship, I run my own track: designing and shipping multi agent products end to end, from architecture to deployment to the outreach that finds them users. I care about systems that reason in steps, recover from their own mistakes, and ship clean enough to hand to a real client.

- 🔭 **Currently building:** Multi-agent research and outreach systems on LangGraph, plus a self-healing coding assistant evaluated with an LLM-as-Judge pipeline.
- 🌱 **Currently deepening:** Agentic AI design patterns, sandboxed code execution, and production grade RAG.
- 🌍 **Currently growing:** A freelance AI engineering practice serving international clients, with active outreach into the Qatar market.
- 🤝 **Beyond code:** Vice Head at the FAST Community Service Society, and a volunteer teacher for underprivileged students.
- 🎯 **Right now:** Actively looking for AI/ML engineering internships and entry-level roles, locally and remote, worldwide.

### 🚀 Featured Projects

#### 🔍 [Aetheris](https://aetheris-deep-research.vercel.app)
A live multi-agent deep research assistant built on LangGraph. Sequential and parallel agent workflows share state to plan a research query, gather sources, and synthesize a structured report.
- **Stack:** Python, LangGraph, Gemini API, Next.js
- **Highlight:** Fully deployed and publicly usable, not just a notebook demo.

#### 📧 [OutreachNode.ai](https://github.com/MugheesTayyab/outreachnode-ai)
A B2B cold email multi-agent system built around a Prospecting Agent, a Context Agent, a Copywriter Agent, and a self-correcting Proofreader Agent that reviews and refines its own output before send.
- **Stack:** Python, LangGraph, LLM APIs
- **Highlight:** Closed-loop self-correction instead of a single pass generation chain.

#### 🤖 [Cold Emailer Multi-Agent System](https://github.com/MugheesTayyab/cold-emailer-agents)
The internal agent pipeline that powers personalized outreach at scale, built during my time at NCRA-UAV.
- **Stack:** Python, LangGraph
- **Highlight:** Each agent owns a single responsibility, making the pipeline easy to debug and extend.

#### 📄 ATS Resume Checker
A practical web app built for a real HR use case: upload a resume, get an ATS compatibility score and structured feedback.
- **Stack:** Next.js, FastAPI, Gemini, PostgreSQL
- **Highlight:** Designed for an actual non-technical end user, not just a portfolio piece.

#### 🧩 [NexusBrief](https://github.com/MugheesTayyab/nexusbrief)
A desktop application demonstrating depth in classical software architecture alongside AI work.
- **Stack:** Java, JavaFX, SQL Server
- **Highlight:** Implements 8 Gang of Four design patterns across an 11 table relational schema.

#### 🛡️ AgentEval — *in active development*
My deep-dive flagship project: a self-healing multi-agent coding assistant.
- **Stack:** Python, Docker, FastAPI
- **Highlight:** Uses an LLM-as-Judge evaluation loop with Docker sandboxing, benchmarked against HumanEval.
